FOR THE PATEINT... BY AN ENGINEER... MAR 25, 2022

It is probably not my first project that is based on the use of Aurdino but,I can say this can be a Life Saver for many pateint.It will be an interesting solution for children, older people and those who have Health Problem's.So, let's have some talk about "HOW.. WHEN..WHY.."

Actually, this Idea came in my when I was in Second Year of My B-tech.I thought that their are many patient who live alone at home.And in case of Emergency they need help as soon as possible.So, they use to call someone to help via call,but he/she can call only one at a time.And in that condition the Patient is not able to call each and everyone of his/her family member.So,here the role of my Project comes...

FEATURES OF CALL-MEE..

  • ALLOW USER TO CALL TOP 5-10 EMERGENCY NUMBER SAVED IN THE SYSTEM

  • USER CAN CALL ALL OF THEM WITH A SINGLE CLICK

  • USER CAN ACTIVATE THE SYSTEM BY USING THE WRISTHAND SWITCH

  • USERS VITAL READINGS WILL BE SHARED VIA MESSAGE

  • NEAREST HOSPITAL WILL RECIEVE AN EMERGENCY CALL VIA SYSTEM FOR HELP

  • " NO PATIENT WILL DIE FOR NOT GETTING HELP ON TIME "

    LET'S DO IT...

    STEP 1:- Sim Module

    I planned that I would start creating this project from creating a prototype on a breadboard. After a few minutes of browsing projects with GSM modules, I decided to order the sim900l module, because it is simply and support all sim.

    STEP 2:- Connection and Power

    I connected the Arduino, GSM module, resistors and transistor to the board and connected all components accordingly.It requires a 5V power supply. I will use a battery from an old smartphone. After connecting the wires from the power supply, the LED should blink every second, which means that the module cannot connect to the network.

    STEP 3:- Inserting a Sim Card

    Now I put a sim card in it and the LED flashes every 3 seconds, informing that the module is connected to the network . After uploading the sketch and entering "ATD+yyXXXXXXXXX;" command, the module should make the call. Then I connected three switches to the Arduino, the first to choose the number, the second to start the call and the third to end the call. I added a few lines to the earlier mentioned program and uploaded it to Arduino.yy - your area code XXXXXXXXX - your number You can Check all AT Commands<--Click here-->
